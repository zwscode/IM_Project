## MyIMProject

#项目描述
使用JAVA实现了一个类似QQ的聊天软件，用户可以进行  
1.注册  
2.添加/删除好友
3.发送小图片  
5.加入某个群组，并进行群聊  
6.离线消息转发  
等操作

#数据存储
服务器端的数据都存储在mysql中
